# Assignment 2 - Shark Species Database (add on)

Assignment 2

Shark Species Database 

this web app displays different shark species. of which you can view some sharks within 
that species and see their length. theres an about page that displays the apps authors information 
and a contact section if you need to contact the author.

As part of assignent 2, there were a few additions to the website. The homepage now shows some statistics. In the dashboard, you can create and delete categories. for each category there is a folder button, to open the contents, and a delete button, to delete the category. there is also a feature to search for sharks within the categories, and it shows what category its in. beside the search prompt is a reset button to reset the prompt and return. in the categories contains the sharks. the user can now create a shark by adding a title and length, they can also edit or delete any current sharks. upon deleting a shark an alert pops up to ask the user if they really want to delete it. the user can sort the sharks by either title or length. they can be ordered in ascending or descending order. the about page has more statistics on the webpage along with icons.
